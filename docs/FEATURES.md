# Features List

HotelPro is packed with features tailored for distinct user roles.

## 1. Customer Experience (Public)
- **"Magical Ledger" Menu**: Beautifully animated vellum-and-ink interface.
- **Master Waiter AI**: An elite AI butler persona for menu guidance and table-side assistance.
- **Neural Voice**: Real-time voice feedback using ElevenLabs for order status.
- **Order Tracking**: Live ink-drawn timeline of the order's journey.

## 2. Waiter Dashboard
- **Visual Floor Plan**: Real-time table occupancy and status tracking.
- **Elite Ordering**: High-velocity interface for waiter-led order entry.

## 3. Kitchen Dashboard (KDS)
- **Live Order Feed**: Real-time display of incoming items.
- **Production Control**: Move items through preparation stages with ease.

## 4. Director Dashboards (Manager)
- **Operational Intelligence**: Real-time revenue ticker and fulfillment velocity metrics.
- **Performance Analytics**: Visual heatmaps of best-selling items.
- **System Audit Feed**: Live log of all critical system actions (Orders, Payments, Staff logins).
- **Menu Architecture**: Strategic control over categories, items, and inventory state.

## 5. Cashier Dashboard
- **Billing**: Generate detailed invoices for active tables.
- **Payment Processing**: Record payments via Cash, Card, or UPI.
- **Order Closure**: Finalize orders and release tables.

## 6. Technical Features
- **Arcjet Protection**: Shielding the application from bots and excessive requests.
- **Responsive Design**: Optimized for tablets (ideally for staff) and mobile phones (for customers).
- **Dark/Premium Aesthetic**: A curated "Vellum and Ink" or modern dark theme for a premium feel.
